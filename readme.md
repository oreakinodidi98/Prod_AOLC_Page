## ğŸ“‹ Objectives

The Centre focuses on ten key objectives:

1. **Enhance Leadership Skills** - Comprehensive training in communication, decision-making, problem-solving, and conflict resolution
2. **Promote Self-Awareness and Emotional Intelligence** - Supporting leaders to understand themselves and others better
3. **Foster Ethical and Responsible Leadership** - Instilling integrity, accountability, and ethical behaviour
4. **Develop Strategic Thinking and Vision** - Training leaders to anticipate challenges and form long-term plans
5. **Encourage Innovation and Change Management** - Equipping leaders to drive innovation and navigate change
6. **Build Effective Team Leadership** - Helping leaders motivate, inspire, and manage high-performing teams
7. **Support Continuous Professional Development** - Encouraging ongoing growth and awareness of emerging trends
8. **Expand Networking Opportunities** - Connecting current and aspiring leaders in a collaborative community
9. **Prepare Leaders for Diverse Contexts** - Training leaders for multicultural, remote, and fast-changing environments
10. **Evaluate and Measure Leadership Impact** - Assessing the effectiveness of leadership training and its organisational impact

This site is configured for deployment on **Azure Static Web Apps**.

### Prerequisites

- Azure account
- Azure CLI (optional)
- Git

### Deploy to Azure Static Web Apps

1. Fork or clone this repository
2. Create a new Static Web App in Azure Portal
3. Connect your GitHub repository
4. Configure build settings:
   - **App location**: `/`
   - **Output location**: `/`
5. Azure will automatically deploy on push to main branch
